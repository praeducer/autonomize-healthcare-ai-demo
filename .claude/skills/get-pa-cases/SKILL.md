name: get-pa-cases
description: Use when the user asks what PA cases are available, wants to see the case list, or asks about test scenarios
---

Read `data/sample_pa_cases/README.md` and present the case list as a table.

Then say: Use `/inspect-pa-case <number or name>` to see the clinical data, `/invoke-pa-review <number or name>` to run AI review, or `/invoke-pa-review-all` to review all cases.
