# Reference Data

## ICD-10-CM Codes (2026)

`icd10cm_codes_2026.tsv` — Curated subset of ~100 ICD-10-CM codes covering the demo's clinical scenarios.

**Source:** [CMS ICD-10-CM 2026 Code Descriptions](https://www.cms.gov/medicare/coding-billing/icd-10-codes/2026-icd-10-cm)
**Format:** TSV with columns: `code`, `description`
**License:** Public domain (US government work product)

### Code Families Included

- M54.* — Dorsalgia (back pain)
- M47.* — Spondylosis
- M48.* — Other spondylopathies
- M05.* — Rheumatoid arthritis
- C34.* — Malignant neoplasm of bronchus and lung
- C77.* — Secondary malignant neoplasm of lymph nodes
- E11.* — Type 2 diabetes mellitus
- L57.* — Skin changes due to chronic exposure to nonionizing radiation
- Common codes for demo scenarios (fractures, imaging indications, etc.)
