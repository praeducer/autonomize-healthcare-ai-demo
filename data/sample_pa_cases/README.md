# Sample PA Cases

5 realistic prior authorization test cases created during Phase 1 implementation:

1. **Lumbar spinal fusion** — clear approval (meets all guidelines)
2. **Knee MRI** — straightforward approval (standard diagnostic)
3. **Biologics for RA** — step therapy required (pend for review)
4. **Lung cancer staging PET/CT** — approval with clinical evidence
5. **Cosmetic dermatology** — denial (not medically necessary)

Each case includes ICD-10-CM diagnosis codes, CPT procedure codes, clinical notes, and expected determination.

Cases are structured as FHIR R4 Claim resources with Da Vinci PAS extensions.
